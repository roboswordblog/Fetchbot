# Fetchbot
This is a robot that fetches stuff and brings it back to you.
### This will be split into 3 parts:
  1. What is the purpose of this
  2. What I used to make it
  3. How did I make this
  4. What challenges did I face
  5. What is next for the robot
## The purpose of this
The purpose of this robot is to fetch stuff at a certian location, adjust itself, and pick up an object. Then it is supposed to bring it back to you. This was a very fun project, and it was also not very hard to make
## The materials
1. The parts of the robotic hand comes from and old kit called the adeept kit. I am not using the Adeept driver controller
2. I am using a mini breadboard for wiring the robotic hand to the arduino.
3. Arduino as a microcontroller and a inplace hand driver
4. Raspberry pi as the microprocesser and the head brain
5. l298n as the motor controler
6. 2 motors, to drive the robot around
7. A screwdriver for using with the l298ns screw ports.
8. a lot of wires of all types, both long and short.
9. All the raspberry pi nessiscities
10. A usb cable to connect the raspberry pi to the arduino
11. A battery case for storing the batteries
12. 4 AA batteries
